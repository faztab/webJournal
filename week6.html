Group Project with Others

Summary

Explanation:
For our group project, we decided to create a game using JavaScript that works in the terminals. We designed the game in a way that the index file served as the main controller, handling the general game flow by calling different functions. To ensure reusability and modularity, we utilized classes to set the properties and functions that could be accessed from anywhere in the code. For example, we created classes to manage the player's stats such as health, attack, defense, and name. These classes allowed us to easily access and modify these stats throughout the game. One of the key components of our game was the inquiries file. It contained various functions that took input from the user and returned the result. These functions were extensively used in almost all of the other files, allowing us to gather user input and make decisions based on it. In terms of organization, we made sure to separate different functionalities into separate functions. For example, we had a main combat loop function that handled the core of the combat system. Additionally, we created other functions that were regularly called upon, such as handling inventory management or displaying game menus. Overall, our project went well, and we were able to create a functional game using JavaScript and branching. However, there were some challenges along the way. One difficulty we faced was ensuring proper communication between different files and functions. We had to carefully plan and test the flow of data to avoid any bugs or inconsistencies. Another challenge was managing the complexity of the game. As the codebase grew, it became harder to keep track of all the different functions and their interactions. We had to spend extra time organizing and documenting our code to make it easier for others to understand and maintain. In conclusion, our group project using JavaScript and branching was a success. We were able to create a game that worked in the terminals and had a clear flow of functions and classes. Despite some challenges, we learned a lot about teamwork and coding practices.
