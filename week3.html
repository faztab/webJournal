If Else Statements

Summary

Summary:
If-else statements in JavaScript are used to make decisions in our code. They allow us to execute different blocks of code based on certain conditions. If a condition is true, the code inside the if block is executed. If the condition is false, the code inside the else block (if present) is executed.

Example

Example:
let num = 10;
if (num > 0) {
    console.log("The number is positive.");
} else {
    console.log("The number is non-positive.");
}
In this example, the if-else statement checks if the value of the variable num is greater than 0. If it is, the code inside the if block is executed, resulting in the message "The number is positive." being printed to the console. If the condition is false, the code inside the else block is executed, resulting in the message "The number is non-positive." being printed to the console.

Analogy

Think of if-else statements like making decisions. Just as you might check the weather before going outside, in JavaScript, if-else statements allow you to check conditions and execute different blocks of code based on those conditions.

Misconception

A common misconception is that if-else statements can only have one condition. However, you can have multiple conditions using else if statements, allowing for more complex decision-making in your code.
