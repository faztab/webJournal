Dot Notation

Summary

Explanation:
Dot notation is used in programming to access the properties or methods of an object. It involves specifying the object, followed by a dot, and then the property or method that you want to access. This allows you to retrieve or set the value of specific properties, or perform actions using the methods, of an object.

Examples

Example 1: person.name
Accesses the name property of the person object.

Example 2: person.age = 25
Sets the value of the age property of the person object to 25.

Analogy

Think of dot notation like addressing a letter. The object is the recipient, and the property or method is the specific information/action you want to communicate. Just as you write the recipient's name followed by a dot and specific information on the envelope, in programming, you specify the object followed by a dot and the property/method you want to access.

Misconception

A common misconception is that dot notation can be used with any data type, but it is specific to objects. Dot notation is used to access object properties and methods, not primitive data types like numbers or strings.
